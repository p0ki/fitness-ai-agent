# Roadmap

## Completed

- Dockerized Python 3.11 Telegram polling foundation
- SQLite structured memory and profile/settings storage
- Text logging for meals, workouts, weight, and body measurements
- Daily and weekly summaries from structured logs
- Privacy, export, backup, and user-scoped soft-delete tools
- Food-photo workflow with temporary downloads, AI estimates, and confirmation
- Progress-photo workflow with temporary downloads, visual-only metadata, and safe direct feedback

## Planned

- Equipment and plate inventory
- Exercise preferences and limitations
- Workout, stretching, meditation, and recovery planning
- Simple Telegram reminders
- Unraid deployment documentation refinement

## Later Ideas

- Exercise alternatives and tutorial lookup
- Optional private dashboard
- Optional wearable data import
- Additional export formats and trend visualizations
